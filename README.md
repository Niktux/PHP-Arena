warrior
=======

Requirements
------------
* Virtualbox
* Vagrant
* vagran-hostsupdater 
 `vagrant plugin install vagrant-hostsupdater`

Installation
------------
* Fetch sources and configure development environment
```
git clone git@github.com:Niktux/warrior.git
php composer.phar install
cd env
vagrant up
```

* Open browser at `http:\\warrior.dev`
